# Modelamiento-calidad-de-aire
Proyecto para el modelamiento numérico de la calidad del aire mediante un modelo de dispersión gaussiano.

Este proyecto corresponde al Reto 1 – Modelamiento de Calidad del Aire, cuyo objetivo principal fue integrar, limpiar, organizar y analizar los datos provenientes de varios reportes del SISAAIRE. Cada archivo contenía mediciones diarias de diferentes contaminantes atmosféricos reportados por la estación FACATATIVA – URBANO.

El trabajo realizado consistió en:

Unificación de múltiples archivos CSV en un único conjunto de datos estructurado.

Transformación y estandarización de fechas, asegurando una línea temporal continua.

Organización de la información por día, integrando contaminantes como NO₂, CO, PM2.5, PM10 y SO₂.

Identificación de días completos y días con valores faltantes, sin eliminar información relevante.

Creación de subconjuntos de datos:

Top 10 días con más mediciones.

Primer día disponible organizado.

Primeros 200 registros del dataset (incluyendo valores NaN cuando corresponda).


Exportación automática de todos los resultados procesados a la carpeta exportado, para ser utilizados por los demás integrantes del equipo.


Este repositorio contiene todo el código utilizado, junto con los archivos generados en formato CSV para facilitar el trabajo colaborativo entre los miembros del grupo.
